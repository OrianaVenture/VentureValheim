# Server Side Multiplayer Tweaks

Created by [OrianaVentureMod@gmail.com](https://github.com/OrianaVenture/VentureValheim).

## Introduction

Commonly requested tweaks for dedicated servers, compatible with vanilla clients. This mod is ONLY required on the server!

## Features

* Toggle player Map Position Icons always on or off.

## Installation

This mod is ONLY required on the server! Live changes to the configurations should take immediate effect.
