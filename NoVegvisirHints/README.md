# No Vegvisir Hints

Created by [OrianaVentureMod@gmail.com](https://github.com/OrianaVenture/VentureValheim).

## Introduction

Disables the discovery map pins from Vegvisir runestones in the world. Interacting with boss location runestones and the Hildir map table still adds pins as expected.

## Features

* Disables the discovery map pins from Vegvisir runestones in the world.
* Adds a configuration SetLookDirection to enable the camera turn direction hint from vanilla "no map mode" to all map modes (defaults to true).

## Installation

This mod needs to be on the client to work. Config Syncing is included with Jotunn. Install on the server to enforce the same mod configuration for all players. Live changes to the configurations will take effect immediate effect.
