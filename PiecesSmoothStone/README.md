## Stability

Please note that stone stability in vanilla valheim is very low. To place the stone roof pieces added by this mod you will need to support them from underneath or find another mod that reduces or removes this instability factor.

## Installation

This mod needs to be on both the client and server; the mod will enforce installation. Players without the mod will NOT be able to connect to the server.

If the mod is removed from the server after pieces have been placed they will disappear.
