# No Unlock Spam

Created by [OrianaVentureMod@gmail.com](https://github.com/OrianaVenture/VentureValheim).

## Introduction

Adds the ability to dismiss (with a hotkey) or fully disable the large unlock popup messages when you discover new crafting materials, recipes, pieces, and items.

## Features

Dismiss all queued unlock messages with a configurable hot key (default ``U``). Optionally turn off these messages entirely with the configuration NotificationsEnabled set to false.

Need to check the list of unlocks after dismissing them? Open the in-game compendium and check the ``Message Log``.

## Installation

This mod is client side and changes made to the configurations will only affect your client. Live updates to the configs will take immediate effect.
